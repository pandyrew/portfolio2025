# Lanyard Assets

Please download the following files and place them in this directory:

1. `card.glb` - The 3D model of the card/badge
2. `lanyard.png` - The texture for the lanyard band

You can download these from the Vercel event badge component repository or create your own:

- Edit your card.glb file at: https://modelviewer.dev/editor/
- The lanyard.png can be edited in any image editor
